# 게임 시스템 문서

이 문서는 은하영웅전설과 삼국지 게임의 핵심 시스템을 상세히 설명합니다.

## 은하영웅전설 - 주요 시스템

### 1. 세션 시스템
- 세션당 최대 2000명
- 실시간 1초 = 게임 24초 (24배속)
- 승리 조건 달성 시 종료

### 2. 캐릭터 8가지 파라미터
- 통솔, 정치, 운영, 정보
- 지휘, 기동, 공격, 방어

### 3. 계급 시스템
제국군: 원수(5) → 상급대장(5) → 대장(10) → 중장(20) → 소장(40) → 준장(80)

### 4. 커맨드 포인트
- PCP (정치), MCP (군사)
- 2시간마다 회복 (실시간 5분)

### 5. 함대 종류
- 함대: 18,000척
- 수송함대: 6,900척
- 순찰대: 900척
- 독행함: 1척

### 6. 작전 시스템
- 점령작전, 방위작전, 소탕작전
- 공적 포인트 획득

### 7. 전술 전투
- 실시간 24배속
- 이동, 선회, 공격, 퇴각
- 진형 변경

### 8. 병참 관리
- 행성창고 ↔ 부대창고
- 할당, 재편성, 보충, 수리

### 9. 그리드 시스템
- 공간/성계/항행불능 그리드
- 진영당 최대 300 유닛

### 10. 승리 조건
- 결정적: 인구 90%+, 함선 10:1
- 한정적: 수도 점령
- 국지적: 인구 우위

## 삼국지 - 주요 시스템

### 캐릭터
- 무력, 지력, 정치, 통솔력, 매력
- 관계와 충성도

### 영토
- 도시 통제 및 개발
- 자원 생산, 과세

### 외교
- 동맹, 배신, 혼인, 조공
